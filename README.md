# Sleep-cycle-and-productivity-data-analysis

Sleep Cycle & Productivity Analysis

This project analyzes how sleep habits, lifestyle patterns, and daily routines affect productivity.
The dataset is taken from Kaggle and contains information such as sleep duration, work hours, stress level, mood, exercise level, and more.
